QUBIC - Smart Modular Building Tool
------------------------------------------------------
Thanks for purchasing QUBIC from the Unity Asset Store.
Qubic is a fast and intuitive editor tool for constructing architectural designs in Unity using modular prefabs.
Build buildings, rooms, bridges, and other structures directly in the Editor with grid-based logic - ideal for quick level design, 
visual prototyping, and flexible architectural layout.

Full user guide here: https://docs.google.com/document/d/1dSxqUGTbihdTqLsPRBO3JFuZ1Aij3nN15FFACrcJ5mM/edit?usp=sharing

Also you can click Help icon on Qubic's components to open online help.

QUICKSTART:
------------------------------------------------------
The easiest way to study Qubic and explore how it works - open the demo scene "01 - Basic" and play with it.

To use Qubic from scratch and for your prefabs, do the following:
 * Open Prefab Manager - via the main menu "Tools/Qubic/Prefab Manager".
 * In Prefab Manager create a new prefab database - via the button "New Database". 
   Set fields Cell Size and Cell Height in the created database. 
   To do it, select the database scriptable object in the Project window and set the fields in the Inspector window.
 * Open a demo scene that contains your prefabs (preferable) or just create a new scene.
 * Select prefabs in the demo scene or select prefabs in the Project window. For each prefab assign template in Prefab Manager.
 * In parallel, create QubicBuilder on scene via the main menu "Tools/Qubic/Create Qubic Builder".
 * Assign created database to PrefabDatabase field of QubicBuilder.
 * As you add prefabs to the database, QubicBuilder will automatically rebuild and you will see how your prefabs appear in the building.
 * You can adjust your prefabs selecting them inside Qubic: move/rotate/change bounds.

You can watch the whole process in the video tutorial: https://youtu.be/a1aMIOG8GsU
