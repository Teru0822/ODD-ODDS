# ドキュメント遷移・探索テスト

## テストの目的
AIエージェントが `AGENTS.md` を起点として正しく下層の詳細ルール・ドキュメント（`Docs/...`配下）に到達して情報を引き出せるか検証します。

## 【テスト指示】
1. 頭の中で必ず `AGENTS.md` を起点とし、そこからインデックスを経由して以下の3つの情報を探し出す手順を実行してください。
   - 「private 変数の命名規則はどうなっているか？」
   - 「SceneやPrefabを複数人で編集・変更する際の注意点は何か？」
   - 「タスク完了後の報告フォーマットにおいて、毎回まとめるべき4つの必須項目は何か？」

2. 見つけ出した情報を元に、以下のフォーマットでユーザーへ報告してください。
   - **辿ったルート:** 例 `AGENTS.md` -> `Overview.md` -> `CodingRules.md`
   - **新たに何が分かったか（テスト結果の評価）:** インデックスから迷わず下層の詳細ルールへ到達可能かどうかの総合評価。
   - **確認した内容（得られた回答）:** 上記3つの質問に対する実際の回答要約。
   - **懸念点:** リンク切れなどがあれば報告する。
