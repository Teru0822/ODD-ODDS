# ハブ・リンク整合性チェック

## チェックの目的
ドキュメントのポータルである `AGENTS.md` と `Docs/01_Rules_and_Policies/Overview.md` に記載されているインデックス（目次）が、実際のファイルパスやディレクトリ構成と矛盾していないかを確認します。

## 【チェック手順】
1. `AGENTS.md` を読み、参照先のファイル群が存在するかを確認します。
2. `Docs/01_Rules_and_Policies/Overview.md` に記載されている全リンク先（ファイル）が存在するかを確認します。
3. `Docs/` 配下に存在するドキュメントで、インデックスから漏れている新ファイルがないか探査します。

## 報告
チェック結果として、リンク切れの有無、インデックス未登録のファイル有無をユーザーへ通知してください。
