      - すごろく（ステージ）
      - パック開封（移動手段など）
        - 移動カード（指定された方向に移動、引いたら必ずその場で使う）
        - アイテムカード
      
  - 第二回の会議メモ
    - Boothから3Dモデルカテゴリーを見つける
    - 机上で完結するようにする
    - カード10枚(上限)を机上に置く，上限を超えた場合，交換する
    - カードはAIやUnityassetなどをつかって作る
    - 机上のきんちゃく袋(お金）をクリックすると，ローグライク要素(レベルアップ等)の画面表示
    - 盤面はいつでもズームしてみることができる
    - カードを買い，パックを引いた後，強制で盤面がズームし，移動する

# コンセプト

目標：

- 短期間で作れるようなシンプルなゲーム性
- ワールドは小さく(1部屋)
- ギャンブル要素で中毒性のあるゲームデザイン

**目指す方向性:** 成長要素を追求し、魅力的なゲームループを実現  
  
背景音:
- **BGM:** オルゴール音を基調に、不協和音を含む暗い雰囲気も
- **SE:** 適切な効果音：[Sound Effect Generator](https://soundeffectgenerator.org/ja/sound-effects) を使用

---

**細かい改善要素**
- 持てるアイテム個数を制約する。
- 特典要素としてアチーブメントを実装。
